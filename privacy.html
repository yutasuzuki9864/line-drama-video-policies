プライバシーポリシー (Privacy Policy)

最終更新日: 2026年3月1日

1. 概要

LINE Drama Video（以下「本サービス」）は、LINE風対談ドラマのショート動画を自動生成・投稿する個人利用のコンテンツ制作ツールです。本ポリシーでは、本サービスにおけるデータの取り扱いについて説明します。

2. データの収集

本サービスは、エンドユーザーや視聴者の個人データを一切収集・保存・処理しません。本サービスは単一のクリエイターアカウントの代理としてのみ動作します。

3. 第三者APIへのアクセス

本サービスは、アカウント所有者が付与した OAuth 2.0 認可を使用して以下の第三者APIにアクセスします：

  • YouTube Data API v3 — 動画のアップロードおよびチャンネル分析データの取得
  • TikTok Content Posting API — 動画のアップロード

APIトークンは運営者のローカルマシンにのみ保存され、第三者に送信されることはありません。

4. データの保存

すべてのデータ（動画、シナリオ、認証トークン）は運営者のローカルマシンに保存されます。クラウドストレージや外部データベースは使用しません。認証情報はバージョン管理から除外された secrets/ ディレクトリに保存されます。

5. データの共有

本サービスは、運営者が意図するソーシャルメディアプラットフォームへの動画コンテンツのアップロードを除き、いかなるデータも第三者と共有しません。

6. Cookie

本サービスは Cookie やトラッキング技術を使用しません。

7. 児童のプライバシー

本サービスは13歳未満の児童を対象としていません。児童の個人データを意図的に収集することはありません。

8. ポリシーの変更

本ポリシーは随時更新される場合があります。変更は上記の「最終更新日」に反映されます。
